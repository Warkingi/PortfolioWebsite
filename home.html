Projects

Empires: Red vs Blue

PvP Teams Capture Point Class Customisation Perks RPG-Style Vendors

Developed during the early days of Fortnite Creative 2.0, Empires: Red vs Blue was my most prized UEFN project. Originally Empires was developed under the name of Twilight Hollow and featured 4 players independently fighting for control over the grove. However, adapting the the UEFN Ecosystem was nessecary and Twilight Hollow's gameplay had it's issues. Listening to feedback and getting the chance to engage with players personally, I developed a new streamlined 2.0 update and renamed the project to Empires.

Features

  • Fully customisable player loadouts.
  • Unique perks for individual gameplay.
  • Large detailed map with 7 PoIs.
  • Warfare on an epic scale.

Graveyard RPG

PvE PvP Leveling Quests Class Powers Loot Vault Dungeons RPG

Graveyard was originally developed as a seasonal project but eventually pivoted into an RPG. This was an ambitious project with one thing in mind; how far can we push UEFN when trying to create an MMORPG?

Features

  • RPG Leveling system with stat scaling.
  • Class powers allowing for deeper customisation.
  • Loot vault for storing all player unlocks.
  • Dungeon system allowing for both solo and co-op.
  • PvE and optional PvP flagging.

World of Heroes

RPG PvE PvP Leveling Quests Class Masteries Open World Dungeons

World of Heroes is a passion project that remains unreleased to the public and was developed out of my love for MMORPGs.

Features

  • Full handcrafted open world ready for adventure.
  • RPG Leveling system with stat scaling.
  • Indepth questing system with roleplaying elements.
  • Class masteries allowing for different playstyles.
  • Limited time events.
  • Dungeon system with custom bosses and loot.
  • Challenging arena system with custom bosses.
  • Professions and craftable items.
